# RF-D-Kapi-Sistemi
Arduino, RFID kart okuyucu ve servo motor kullanılarak geliştirilen bu projede, yetkili RFID kart okutulduğunda kapının açılmasını sağlayan akıllı kapı kilidi sistemi tasarlanmıştır. Sistem, maket bir ev üzerinde uygulanmış olup servo motor yardımıyla kapı mekanizması kontrol edilmektedir.
